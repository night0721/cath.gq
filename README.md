
# Cath.gq

Cath.gq is the dashboard for its parent project, [Cath.exe](https://github.com/night0721/cath.exe)


## Run Locally

Clone the project

```bash
  git clone https://github.com/night0721/cath.gq.git
```

Go to the project directory

```bash
  cd cath.gq
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  npm run dev
```
